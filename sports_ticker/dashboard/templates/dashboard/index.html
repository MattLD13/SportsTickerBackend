About

A custom sports ticker built on 6× chained 64×32 HUB75 RGB LED panels (384×32 total). A Flask server runs on an Ubuntu VPS, aggregates live data from a dozen sources, and serves display payloads to one or more Raspberry Pi controllers over the network.
